# Vue SSR server

## Before begin dev
```npm i```

## Start dev server
```npm run start:dev```

## Stop dev server
```npm run stop:dev```

## Configuration

Configuration file should be located in ```./config.yaml```.

### Configuration props:

- host - server listen address
- port - server listen port

### Example
```
host: 127.0.0.1
port: 8080
```

## Logs

Logs located in ```./log``` folder.

Fast show server log: ```npm run log```
